European Social Survey (Round 11) — Filtered Codebook

1. Core Demographics & Geography

  • Variable Name: idno
    • Custom Alias: respondent_id
    • Description: Respondent's unique identification number.
    • Scale Type: Continuous / Discrete ID.
    • Missing Values: None (System generated).
  • Variable Name: cntry
    • Custom Alias: country_code
    • Description: Two-letter ISO country code identifying the country of interview (e.g., FR, DE, GR, IT, PL, ES, SE).
    • Scale Type: Categorical / Nominal string.
    • Missing Values: None.
  • Variable Name: regunit
    • Custom Alias: nuts_level
    • Description: Regional unit level indicating the NUTS structural depth used for sampling within the specific country.
    • Scale Type: Categorical.
    • Missing Values: None.
  • Variable Name: region
    • Custom Alias: nuts_region
    • Description: Subnational region code matching the standard Eurostat Nomenclature of Territorial Units for Statistics (NUTS) classification (typically NUTS 1 or NUTS 2).
    • Scale Type: Nominal alphanumeric code.
    • Missing Values: NA (Not available).

2. Survey Weights

  • Variable Name: dweight
    • Custom Alias: design_wt
    • Description: Design weight adjusted for differences in select probabilities across sampling frames.
    • Scale Type: Continuous numeric.
  • Variable Name: pspwght
    • Custom Alias: post_stratification_wt
    • Description: Post-stratification weight including design weight, calibrated to national population parameters (gender, age, education, and region).
    • Scale Type: Continuous numeric.
  • Variable Name: pweight
    • Custom Alias: pop_size_wt
    • Description: Population size weight used to scale samples proportionally to the actual size of each country's total population.
    • Scale Type: Continuous numeric.
  • Variable Name: anweight
    • Custom Alias: analysis_wt
    • Description: Calculated analysis weight (pspwght $\times$ pweight) required when pooling multi-country samples to prevent small-population nations from being over-represented or distorted by larger nations.
    • Scale Type: Continuous numeric.
