# Resilience Gap Project

An analytics engineering pipeline built using **dbt Core** and **PostgreSQL** to clean, transform, and analyze data from the European Social Survey (Round 11). Party classifications from the PopuList database are also used. Further data wrangling and visualization takes place using Python.

## 📂 Project Structure & Data Pipeline

This repository contains the core dbt transformation layer and the .ipynb notebooks. The raw data itself is kept locally and excluded from version control to comply with file size constraints and data privacy best practices.
